# Dashboard-Ordner — Die Webseite 🖥️🎨

Der Ordner `dashboard/` enthält das Cockpit. Hier arbeiten die Mitarbeiter mit dem **R_agent**.

## 📄 Wichtige Teile

| Datei | Zweck |
|-------|-------|
| `index.html` | Die **Struktur der Seite**: Mit HTML5 gebaut. |
| `style.css` | Das **Design**: Ein modernes Design mit Glas-Effekten und Farben. |
| `app.js` | Die **Steuerung**: Holt Daten vom Server, zeichnet die Karte und zeigt die Gedanken der KI. |

## 🌟 Peak Demo: Wichtige Funktionen
1.  **Weltkarte für Risiken**: Mit **Leaflet.js** und einem dunklen Design.
2.  **Szenario-Steuerung**: Ein Zentrum für Tests der Krisen.
3.  **Gedanken-Panel**: Live-Stream der Gedanken der KI.
