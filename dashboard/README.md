# Dashboard Folder — The Frontend UI 🖥️🎨

The `dashboard/` directory contains the modern, responsive "Cockpit" where procurement managers interact with the **R_agent**.

## 📄 Key Components

| File | Purpose |
|------|---------|
| `index.html` | The **Dashboard Structure**: Built with semantic HTML5. |
| `style.css` | The **Glassmorphism Aesthetic**: Vibrant dark-mode UI with pulsing risk indicators. |
| `app.js` | The **Frontend Orchestrator**: Manages API calls, Map rendering, and SSE event handling. |

## 🌟 Peak Demo: High-Impact Features
1.  **Global Risk Map**: Integrated **Leaflet.js** with a custom CartoDB Dark Matter theme.
2.  **Scenario Controls**: Interactive Stress-Test center for simulation triggering.
3.  **Thought-Trace Panel**: Live stream implementation for AI reasoning transparency.
