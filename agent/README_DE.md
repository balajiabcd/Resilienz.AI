# Agenten-Ordner — Das Gehirn & die Werkzeuge 🧠🧰

Der Ordner `agent/` enthält die Intelligenz für den **R_agent**. Hier finden Daten und Denken zusammen.

## 📄 Bausteine

| Baustein | Zweck |
|----------|-------|
| `tools.py` | Die "Hände" — 8 Python-Funktionen für SQL, Vector DB und Berichte. |
| `llm/` | **Der KI-Schalter**: Mehrere KIs (Gemini & OpenRouter) für mehr Sicherheit. |
| `brain.py` | Steuert die **Denk-Aktion-Schleife**. |
| `prompts.py` | Die "Verfassung" — bestimmt das Verhalten und die Grenzen der KI. |

## 🌟 Peak Demo: Gedanken Senden
Der Agent meldet jetzt seine Schritte live. Während er denkt, nutzt er `emit_thought` für den Server. So kann der Mensch jeden Schritt der KI sehen.
